pub fn set_if_is_none(&self, cpu_id: CpuId) -> Result<(), CpuId>

Sets the inner value of an AtomicCpuId if it’s empty.

The return value is a result indicating whether the new value was written and containing the previous value. If the previous value is empty, it returns Ok(()). Otherwise, it returns Err(previous_value) which the previous value is a valid CPU ID.
