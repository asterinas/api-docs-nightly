pub struct LockedXArray<'a, P, M = NoneMark, G = PreemptDisabled>
where
    P: NonNullPtr + Send + Sync,
    G: SpinGuardian,
{ /* private fields */ }
Expand description

The locked XArray which obtains its inner spinlock.

The locked XArray is able to create CursorMut and do mutable operations. There can only be one locked XArray at the same time.
