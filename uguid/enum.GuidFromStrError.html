Enum GuidFromStrError

pub enum GuidFromStrError {

    Separator(u8),
    Hex(u8),
}
Expand description

Error type for Guid::try_parse and Guid::from_str.

Variants§

§

Length

Input has the wrong length, expected 36 bytes.

§
