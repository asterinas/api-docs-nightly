#[repr(C, align(8))]
pub struct AlignedBytes<const N: usize>(pub [u8; N]);
Expand description

Helper to 8-byte align the underlying bytes, as mandated in the Multiboot2 spec.

With this type, one can create manual and raw Multiboot2 boot information or just the bytes for simple tags, in a manual and raw approach.
