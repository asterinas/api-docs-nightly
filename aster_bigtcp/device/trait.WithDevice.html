Expand description

A trait that allows to obtain a mutable reference of Device.

A Device is usually protected by a lock (e.g., a spin lock or a mutex), and it may be stored behind a shared type (e.g., an Arc). This property abstracts this fact by providing a method that the caller can use to get the mutable reference without worrying about how the reference is obtained.
