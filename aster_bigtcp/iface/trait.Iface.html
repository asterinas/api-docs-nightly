Expand description

A network interface.

A network interface (abbreviated as iface) is a hardware or software component that connects a computer to a network. Network interfaces can be physical components like Ethernet ports or wireless adapters. They can also be virtual interfaces created by software, such as virtual private network (VPN) connections.

pub fn bind( self: &Arc<Self>, config: BindPortConfig, ) -> Result<BoundPort<E>, BindError>

Binds a socket to the iface.

After binding the socket to the iface, the iface will handle all packets to and from the socket.

If BindPortConfig::Ephemeral is specified, the iface will pick up an ephemeral port for the socket.

FIXME: The reason for binding the socket and the iface together is because there are limitations inside smoltcp. See discussion at https://github.com/smoltcp-rs/smoltcp/issues/779.
