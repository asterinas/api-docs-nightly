Extension to be implemented by users of this crate.

This should be implemented on an empty type that carries no data, since the type will never actually be instantiated.

The purpose of having this trait is to allow users of this crate to inject multiple types without the hassle of writing multiple trait bounds, which can be achieved by using the types associated with this trait.
