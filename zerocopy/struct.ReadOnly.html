Struct ReadOnly 

Source 
pub struct ReadOnly<T: ?Sized> { /* private fields */ }
Expand description

A read-only wrapper.

A ReadOnly<T> disables any interior mutability in T, ensuring that a &ReadOnly<T> is genuinely read-only. Thus, ReadOnly<T> is Immutable regardless of whether T is.

Note that &mut ReadOnly<T> still permits mutation – the read-only property only applies to shared references.
