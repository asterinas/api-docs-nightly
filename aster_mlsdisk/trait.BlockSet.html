Expand description

A fixed set of data blocks that can support random reads and writes.

§Thread safety

BlockSet is a data structure of interior mutability. It is ok to perform I/O on a BlockSet concurrently in multiple threads. BlockSet promises the atomicity of reading and writing individual blocks.
