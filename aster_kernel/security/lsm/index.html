The Linux Security Module (LSM) framework.

LSM lets the kernel route security-sensitive operations through a stack of built-in policy modules. Each module can implement shared hook traits and inspect common hook contexts before allowing or rejecting an operation.

This module defines the common LSM traits and hook contexts shared by built-in modules such as capability and yama. Module selection follows the lsm= and legacy security= kernel command-line parameters.
