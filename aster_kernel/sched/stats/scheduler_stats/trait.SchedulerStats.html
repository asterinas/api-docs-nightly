fn nr_queued_and_running(&self) -> (u32, u32)

Returns a tuple with the number of tasks in the runqueues and the number of running tasks.

We decided to return a tuple instead of having two separate functions to avoid the overhead of disabling the preemption twice to inspect the scheduler.
