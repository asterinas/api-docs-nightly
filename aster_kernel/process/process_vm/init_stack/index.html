The init stack for the process.

The init stack is used to store argv, envp, and auxiliary vectors. We can read argv and envp of a process from the init stack. Usually, the lowest address of the init stack is the highest address of the user stack of the main thread.

However, the init stack will be mapped to user space and the user process can write the content of the init stack, so the content reading from the init stack may not be the same as the initial one.
