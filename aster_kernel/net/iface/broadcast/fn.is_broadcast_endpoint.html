pub fn is_broadcast_endpoint(endpoint: &IpEndpoint) -> bool
Expand description

Determines if a given IP endpoint’s address is a known broadcast address.

IPv6 has no broadcast; multicast (ff00::/8) handles fan-out instead and is intentionally not covered by this function.
