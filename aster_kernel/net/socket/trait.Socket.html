fn recvmsg( &self, writer: &mut dyn MultiWrite, flags: SendRecvFlags, ) -> Result<(usize, MessageHeader), Error>

Receives a message from the socket.

If successful, the io_vecs buffer will be filled with the received content. This method returns the length of the received message, and the message header.
