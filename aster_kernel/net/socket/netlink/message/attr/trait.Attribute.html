fn read_from( header: &CAttrHeader, reader: &mut dyn MultiRead, ) -> Result<ContinueRead<Self>, Error>
where Self: Sized,

Reads the attribute from the reader.

This method may return a ContinueRead::Skipped or a ContinueRead::SkippedErr if the attribute is not recognized or not valid. In that case, however, it must still skip the payload length (excluding padding), as if the attribute were parsed properly.
