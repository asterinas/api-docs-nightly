Trait defining the write behavior for a collection of VmWriters.

Required Methods§

Source

fn write( &mut self, reader: &mut VmReader<'_, Infallible>, ) -> Result<usize, Error>

Writes the exact number of bytes required to exhaust writer or fill self, accumulating total bytes read.

If the return value is Ok(n), then n should be min(self.sum_lens(), reader.remain()).

§Errors

This method returns Errno::EFAULT if a page fault occurs. The position of self and the reader is left unspecified when this method returns error.
