Required Methods§

Source

fn read( &mut self, writer: &mut VmWriter<'_, Infallible>, ) -> Result<usize, (OstdError, usize)>

Reads the exact number of bytes required to exhaust self or fill writer, accumulating total bytes read.

If the return value is Ok(n), then n should be min(self.sum_lens(), writer.avail()).

§Errors

This method returns OstdError::PageFault if a page fault occurs, along with the number of bytes copied before the error occurs. When an error is returned, both self and writer are advanced by the returned byte count.
