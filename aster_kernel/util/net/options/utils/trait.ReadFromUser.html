Expand description

Create an object by reading its C counterpart from the user space.

Note that the format of a value in the user space may be different from that
in the kernel space. For example, the type of a boolean value in the kernel is expressed as bool, whereas that in the user space is i32.

In addition, since the user space is not trusted by the kernel, values read from the user space must be validated by the kernel.
