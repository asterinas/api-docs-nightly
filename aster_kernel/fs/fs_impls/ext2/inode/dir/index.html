Directory operations for ext2 inodes.

Directory state is stored as ext2 directory entries inside ordinary inode data blocks. This module preserves the VFS-visible directory semantics for lookup, creation, hard links, rename, removal, and iteration while keeping directory entries and link counts consistent.
