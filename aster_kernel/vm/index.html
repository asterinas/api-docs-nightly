Virtual memory (VM).

There are two primary VM abstractions:

  • The VMAR (used to be Virtual Memory Address Region, now an orphan initialism) represents the entire virtual address space of a process;
  • The VMO (Virtual Memory Object) is a set of logically contiguous memory frames that can be mapped into one virtual address range. Frames in a VMO can be non-contiguous in physical memory.
