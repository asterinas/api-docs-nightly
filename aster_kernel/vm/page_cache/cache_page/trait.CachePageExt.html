Expand description

Convenience operations on a CachePage handle.

Implemented for every CachePage, this gives access to the page lock, the per-page wait queue, and lifecycle helpers used during page commit.

Clears writeback state for a page whose writeback has completed or was canceled.

Callers must invoke this method only to finish a writeback operation previously started with LockedCachePage::set_writing_back, either from its completion callback or while handling submission failure.
