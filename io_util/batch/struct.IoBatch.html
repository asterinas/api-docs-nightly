Struct IoBatch

pub struct IoBatch { /* private fields */ }
Expand description

A batch of IoCompletion records.

Used as an out-parameter on async-submission APIs: callers create an IoBatch, pass &mut to one or more submission calls, then call wait_all to block until every completion record in the batch has completed.

IoBatch does not track readiness. A completion record is counted by len as long as it is stored in the batch, even if calling IoCompletion::wait on it would return immediately.

Waits for every completion record in the batch.

All completions are waited on even if an earlier one fails. The first observed error is returned.
