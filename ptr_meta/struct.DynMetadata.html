Struct DynMetadata

pub struct DynMetadata<Dyn: ?Sized> { /* private fields */ }
Expand description

The metadata for a trait object.

This struct wraps a pointer to a vtable (virtual method table) which contains all of the necessary information to manipulate the concrete type stored inside of the trait object:

  • The size and alignment of the concrete type
  • A function pointer to the type’s drop_in_place impl
  • Function pointers for each method in the concrete type’s trait implementation

Providing a type argument that is not a dyn trait object is possible, but does not correspond with a meaningful type.
