Module release 

Source
Expand description

FUSE_RELEASE releases an open file handle, and FUSE_RELEASEDIR releases an open directory handle.

Both request bodies contain ReleaseReq for the handle named by the target inode and ReleaseKind selects the opcode. Successful replies do not carry a payload.
